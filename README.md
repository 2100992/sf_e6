Для того, что бы развернуть проект, нужно выполнить следующее:

  -установить docker и docker compose
  
  -`git clone https://github.com/2100992/sf_e6.git`
  
  -`cd sf_e6`
  
  -`docker-compose build`
  
  -`docker-compose up -d`
